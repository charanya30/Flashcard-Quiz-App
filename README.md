# Flashcard Quiz App

A simple flashcard quiz application built with HTML, CSS, and JavaScript.

## Features

- Question on the front and answer on the back.
- Navigate cards with Previous and Next.
- Add, edit, and delete flashcards.
- Clean mobile-friendly interface.
- Stores cards locally in browser storage.

## Usage

1. Open `index.html` in a browser.
2. Use the form to add a flashcard.
3. Click `Show Answer` to reveal the answer.
4. Double-click the current card text to edit it.
